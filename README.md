# 📊 Retail Sales Insights Pipeline (PySpark + Streamlit)

## 🚀 Project Overview
This project demonstrates an end-to-end data pipeline built using PySpark to process and analyze retail sales data. The pipeline performs data cleaning, feature engineering, and generates key business insights. The results are visualized using a Streamlit dashboard.

---

## 🎯 Key Features
- Data ingestion from CSV files
- Data cleaning and preprocessing
- Feature engineering (total_price, date transformation)
- KPI generation:
  - Total Revenue
  - Category-wise Sales
  - City-wise Sales
- Interactive dashboard using Streamlit

---

## 🛠️ Tech Stack
- Python
- PySpark
- Streamlit
- Pandas
